# COSMOS
**Cryptographic Omnivalent Strategic Module to Outmaneuver Securly**

COSMOS is a modular, browser-based platform that brings together games, media, and interactive tools into a single, unified web experience. Designed to be fast, lightweight, and extensible, COSMOS acts as an all-in-one digital hub built to allow students unrestricted entertainment.

Check out the COSMOS site [here](https://ethanytangcodes.github.io/cosmos)!
---

## Features

### Massive Game Library
- 200+ browser-playable games    
- No installs required — runs directly in the browser  
- Optimized for low-latency and fast loading  

### Gameboy Support  
- Save states and customizable controls  
- Clean, distraction-free emulator interface
- Advanced emulator features such as frameskip and cheats  

### Download the file
- Can't be seen by securly 
- Simple, one-click access  
- Single file makes navigation easy  

### Music Player
- Built-in audio player with liked songs and localstorage
- Lyrics so you can sing along  
- Background playback while browsing games   

### AI-Chat
- AI Chat featuring GPT-5 nano
- Designed for interactive and creative use
